--- layout: guide guideName: Using the serial console for debugging Yellow (Linux) ---

Connecting to Yellow via serial console can be helpful for troubleshooting, e.g. to export a log.

  1. Make sure JP1 is set to UART.
  2. On your PC, open a terminal.
  3. List the USB port numbers:
    ls /dev/ttyUSB*.
  4. Setup a USB connection: Yellow: USB-C, PC: USB type A.
  5. List the USB port numbers again. The new number is for Yellow.
    ls /dev/ttyUSB*.
  6. Start the screen application and point it to this USB port. E.g.:
    sudo screen /dev/ttyUSB0 115200.

    Note: Screen typically doesn't display anything on startup. You see only the cursor on the top left of the window.
  7. Hit the Enter key until prompted for credentials.
    • Homeassistant login: root
    • No password is required. Hit the Enter key.
  8. To print logs, enter one of the following commands:
    • To print the supervisor logs, type
      ha supervisor logs
    • To print out the network info, type
      ha network info
  9. To save the boot log into a file, perform the following steps:
    1. Power off Yellow.
    2. Perform steps 1–6.
    3. Power up Yellow.
    4. To save the bootlogs, type
      Ctrl+a :hardcopy -h /tmp/boot.log
