--- layout: guide guideName: "Symptom: Installation on Yellow kit: yellow LED keeps blinking" ---

Symptom

During installation on a Yellow Kit, the yellow LED keeps blinking in a heartbeat pattern.

Cause

If this LED pattern persists during the installation, and the LED never changes to solid on, this indicates an issue with the installer. It may not be able to connect to the internet.

Resolution

You can install Home Assistant OS directly onto the internal eMMC using the `rpiboot` boot method. Follow this steps to do so:

  1. Download the Yellow operating system.
    • Go to the operating system page on github and download the latest version of the `haos_yellow-xx.x.img.xz` file.
  2. Reinstall the operating system on Yellow.
    • Open the procedure on how to reinstall the operating system.
    • Follow the procedure Option 2: Reinstall Home Assistant OS using rpiboot up to step 8.
    • In step 9, use the downloaded image instead of the image provided by the Raspberry Pi Imager.
    • Follow the rest of the steps to finish reinstalling the Yellow operating system.
  3. Your system should now start up.
