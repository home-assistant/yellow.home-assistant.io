This guide assumes you have the following setup:

  • Home Assistant Yellow up and running.
  • Home Assistant OS installation.
  • Multiprotocol support disabled: Refer to the Disable multiprotocol support procedure.
  • This guide installs a firmware which supports only Thread! If you use your Yellow to control Zigbee devices, you will no longer be able to control them after installing the Thread firmware.
---

This guide outlines the steps for configuring the Home Assistant Yellow for dedicated Thread support. This setup facilitates the connection of Matter-over-Thread devices to your network.
