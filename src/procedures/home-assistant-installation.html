--- tags: - poe - powersupply order: 4 procedures: - title: Installing Home Assistant Software on Kit - steps: - title: Install Raspberry Pi Imager image: rpi_imager.png content: |
  • Install imager on your your computer as described under https://www.raspberrypi.com/software/
- title: Insert the USB drive into computer. - title: Open Raspberry Pi Imager. image: start_rpi_imager.png - title: Choose Operating System. image: rpi_choose_ha_os.gif content: |
  • Click Choose OS
  • From the dropdown menu, select Home Assistant Operating System.
- title: Choose Storage. image: rpi_choose_storage.png content: |
  • From the dropdown menu, select your USB drive.
- title: Write. image: rpi_choose_write.png content: |
  • Click the Write button to start the process.
  • Wait for the Home Assistant Operating System Installer to be written to the USB drive.
- title: Eject the USB drive. - title: Insert the USB drive. image: plug-usb-flash.jpeg content: |
  • Insert the USB drive with the Home Assistant Operating System Installer into your Home Assistant Yellow Kit.
- title: Connect the Ethernet cable. image: plug-usb-flash-ethernet.jpeg - title: Connect the power supply. image: plug-usb-flash-ethernet-power.jpeg content: |
  • If you are not using PoE, Connect the power supply to Home Assistant Yellow Kit.
- title: Watch. image: led-patter-boot.webp content: |
  • Wait for Home Assistant Yellow to start booting from the USB drive.
  • During boot, the Yellow LED blinks in a heartbeat pattern.
  • Once the installer is ready, the yellow LED is on constantly.
  • Installation will start automatically after 2 seconds.
- title: Wait. image: led-patter-install.webp content: |
  • Wait for Home Assistant Yellow to finish installing the Home Assistant Operating System.
  • The yellow LED starts to blink steadily.
  • Once the installer finished installing Home Assistant Operating System, only the red LED will be on.
- title: Disconnect everything. image: plug-nothing.jpeg content: |
  • Remove the USB flash drive.
  • Remove the power cable.
- title: Reconnect power. image: plug-ethernet-power.jpeg content: |
  • Reinsert the Ethernet cable.
  • Reinsert the power cable to start Home Assistant (if PoE is not used).
- title: Get started with Home Assistant. image: ha_onboarding.png content: |
  • Initial startup may take longer - depending on your internet connection.
  • Visit http://homeassistant.local:8123 in your Browser to access the device Web frontend.
  • The Web frontend guides you through the initial onboarding.
  • For further information, refer to the instructions provided under https://www.home-assistant.io/getting-started/onboarding/.
